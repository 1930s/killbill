Requirements

Runtime

  • Java JRE: killbill has been tested against OpenJDK6 (1.6.0_23), OpenJDK7 (1.7.0_147-icedtea) and Oracle JDK7 (Oracle JDK7)
  • Container (optional): killbill has been tested against Jetty 7.5.1 (Servlet ~3.0), Jetty 8.1.3 (Servlet 3.0) and Tomcat 7.0 (Servlet 3.0). JBoss AS 7.1 (Java EE 6, full profile) support is experimental.
  • MySQL database: 5.5 or later recommended
  • Kill Bill should not have any other system dependency, although it was only tested under Ubuntu 10.0.4, Ubuntu 12.0.4 and Mac OSX 10.6.8

Compile

  • Java SDK: killbill has been compiled and tested against OpenJDK6 (1.6.0_23), OpenJDK7 (1.7.0_147-icedtea) and Oracle JDK7 (Oracle JDK7)
  • Maven: 3.0.3 or later preferred, but 2.2.1 (and later) is also supported

Dependencies

Library

Note: Kill Bill uses maven and Maven Central to handle dependencies management.

  • config-magic
  • Guava, several of Google's core libraries
  • Jackson, a Java library for processing JSON data format
  • jdbi, a SQL convenience library for Java
  • JSR-330, Dependency Injection for Java (Kill Bill currently still has an explicit dependency on Google Guice, but shouldn't require it)
  • mogwee-executors
  • slf4j, a Simple Logging Facade for Java
  • jmustache, a Java implementation of the Mustache templating language
  • org.apache.commons:commons-email
  • Joda-Time, library for advanced date and time in Java
  • org.antlr:stringtemplate

Container

  • JSR 311 JAX-RS: The Java API for RESTful Web Services
